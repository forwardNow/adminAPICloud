# 系统管理 —— APICloud

## 1. 介绍

使用 APICloud 开发 App。

## 2. 真机调试

* 官网
  * 注册 APICloud 账号
  * 创建 APP
* 本地
  * 使用 SVN 客户端检出代码
  * 在 vscode 中搜索并安装 “appcloud” 插件
* 手机
  * 在手机微信上通过扫描安装官方 [AppLoader](https://docs.apicloud.com/Download/download)
  * 设置“通过 USB 共享网络”，使手机可访问电脑
* 调试
  * vscode：“apicloud:启动wifi服务” -> “apicloud:wifi全量更新”
